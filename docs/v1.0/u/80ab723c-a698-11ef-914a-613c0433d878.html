Category Description

These are non-exclusive grouping of CMIP variables (e.g. monthly time slices of the baseline variables). Variables can belong to more than one group.



AttributeValue
justificationThis group of variables aggregates the primary variables at the vegetation type level. These variables can substitute a series of variables used in CMIP6. For example, gppVgt will contain the information previously given by gppTree, gppGrass, and gppShrub. Moreover, this group of variables could be used as primary variables in assessing phenology simulated by models and the relative impact for future projections at the vegetation type level.
