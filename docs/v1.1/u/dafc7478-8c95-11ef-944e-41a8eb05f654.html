justificationCloud variables of interest to the Southern Ocean BGC to Clouds opportunity. We want to understand how natural aerosol interact with cloud and radiation, and for this we need detailed knowledge of both the marco and micro-physical cloud state. We have requested variables that include the basic cloud physical parameters as well as fields derived from COSP in order to best compare to satellite products.
