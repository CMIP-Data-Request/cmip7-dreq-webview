Model Intercomparison Project



AttributeValue
mip_abstractThe Ocean Model Intercomparison Project (OMIP) aims to provide a framework for evaluating, understanding, and improving the ocean, sea-ice, tracer, and biogeochemical components of global climate and Earth system models contributing to the Coupled Model Intercomparison Project Phase 7 (CMIP7). Ocean and sea ice models not part of a coupled system are also welcome to participate. Existing papers define OMIP and detail protocols both for simulating global ocean/sea-ice models with inert chemical and biogeochemical tracers and for analyzing their output.
