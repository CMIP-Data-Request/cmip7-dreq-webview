Category Description

These are non-exclusive grouping of CMIP variables (e.g. monthly time slices of the baseline variables). Variables can belong to more than one group.



AttributeValue
justificationThis variable group contains a deeper set of variables needed for more complex agricultural modeling and impacts applications. The additional variables allow for better representation of drought stress, water requirements, and extreme impacts. Many of the top agricultural models (e.g., in the Agricultural Model Intercomparison and Improvement Project, AgMIP) have the ability to utilize this additional information along with more complex process representation and parameterizations within their models.
