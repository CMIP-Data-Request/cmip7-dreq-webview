AttributeValue
justificationThese are the lower priority variables attached to the synoptic systems and impacts opportunity. These variables are high frequency (daily or 6 hourly instantaneous) versions of existing fields but on different pressure levels. hur700 is useful for looking at the dynamical controls on clouds associated with synoptic systems, hur100 is useful to distinguish between different tropical systems, hur850 and h500 can be used to detect monsoon low pressure systems and examine convection/MCS and frontal environments. zg700 and zg925 (along with the existing 850 hPa level version can be used to classify various types of low-pressure systems. ua200 and va200 along with the 250 hPa version of these fields can be used to compute deep layer shear.
