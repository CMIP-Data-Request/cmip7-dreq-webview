Category Description

These are non-exclusive grouping of CMIP variables (e.g. monthly time slices of the baseline variables). Variables can belong to more than one group.



AttributeValue
justificationGCOS has defined 7 essential climate sea ice variables (paper by Lavergne et al, BAMS 2022). CMIP output will be useful to evaluate observational products with respect to these variables.
nameseaice_gcos_ecv
notesAlbedo was not included as part of the CMIP7 data request. Instead, we ask for the incoming and reflected SW flux terms that allows to calculate surface albedo. We include both fluxes from the atmosphere model and from the sea ice model to allow to calculate surface albedo over the grid cell or over only the sea ice-covered part of the grid cell.
titleSea ice Essential Climate Variables (ECVs) for Global Climate Observing System (GCOS)
