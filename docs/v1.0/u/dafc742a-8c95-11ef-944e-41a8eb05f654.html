Category Description

These are non-exclusive grouping of CMIP variables (e.g. monthly time slices of the baseline variables). Variables can belong to more than one group.



AttributeValue
justificationThese variables are required for global glacier assessments which include polar glaciers (including peripheral glaciers in Greenland and (Sub-)Antarctica that are decoupled from the ice sheets) and mountain glaciers, as identified in the Randolph Glacier Inventory (RGI). This variable group also supports the WCRP CliC GlacierMIP activities. For details on use cases, please see the related opportunity: "ID 35 Glacier changes, drivers, and impacts." The variables required are at fixed, daily, and monthly temporal resolutions.
